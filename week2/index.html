RTX 3000 Series GPUs

RTX 3070

Nvidia GeForce RTX 3070 is set to launch at Oct 15,2020. It contains 17.4 billion transistors with a die size of 392.5 mm2. The base core clock of the GPU is 1500 MHz where as the boost clock is 1730 MHz and the memory speed is 14000 MT/s with video memory size of 8GBs. The TDP of the GPU is set at 220 watts and it will be priced at $499 USD.
